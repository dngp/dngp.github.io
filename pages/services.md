# Services

This page provides links to the growing number of open source services I host.

## Public services

- [Freetar](https://tabs.adast.dk): Frontend for ultimate-guitar.com
- [PeerTube](https://tv.adast.dk): Video/stream platform (registration invite-only)
- [SearXNG](https://search.adast.dk): Metasearch engine ([short URL](https://s.adast.dk))
- [SimplyTranslate](https://st.adast.dk): Frontend for multiple translation websites

## Invite-only

- Email: Get your @adast.dk mail address 
- XMPP: The universal messaging standard
- [Forgejo](https://git.adast.dk): Software forge
- [MediaMTX](https://live.adast.dk): Low-latency live streams
- [Mumble](mumble://mumble.adast.dk): Low-latency voice chat
- [Nextcloud](https://cloud.adast.dk): Cloud storage, calendar, contacts
- [Radicale](https://cal.adast.dk): Calendar and contacts

Check the [documentation](https://man.sr.ht/~adamski/adast.dk-docs) for setup guides.<br>
